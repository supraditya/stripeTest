# stripeTest
A payments portal test project developed using React, NodeJS and Stripe API

To deploy locally:

 1. Create a stripe account. Set your secret key in index.js in stripebackend folder
 2. Do the same with the public key but in app.js in stripefrontend folder (Do this securely via .env)
 3. Start servers using "npm start" in both stripefrontend and stripebackend folders using separate terminals
 4. You're good to go!
